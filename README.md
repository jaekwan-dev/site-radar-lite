# Site Radar Lite - 건축현장 타겟 관리 도구

건설업계 영업을 위한 전국 신도시/택지개발지구 건축현장 정보 조회 및 관리 시스템입니다.

## 🚀 주요 기능

### 📊 통계 대시보드
- 전체 현장 수, 착공 현장, 6개월 내 준공 예정 현장 통계
- 클릭 시 해당 조건으로 자동 필터링

### 🔍 신도시 현장 조회
- 전국 신도시/택지개발지구 건축허가 및 착공 현장 검색
- 준공예정일, 상태별 필터링 및 정렬 기능
- CSV/Excel 다운로드 지원

### 📝 수동 입력
- 직접 발굴한 현장 정보 수동 등록
- 유효성 검사 및 실시간 피드백

### 🎯 타겟 현장 관리
- 관심 현장을 타겟 리스트로 관리
- 일괄 다운로드 및 삭제 기능
- 로컬 스토리지 자동 저장

### 🔍 상세 정보 조회
다양한 공공데이터 API를 연계하여 종합적인 현장 정보를 제공합니다:

#### KISCON 공공데이터 API 연계
- **건설업체 상세정보**
  - 사업자등록번호, 대표자명, 업체 주소, 등록업종
  - 건설업 등록증 정보(등록번호, 등록일자, 유효기간)
  - 최근 3년간 공사실적(공사명, 발주기관, 계약금액)
- **공사대장 통계**
  - 지역별/업종별 월간 통보 건수 및 계약금액 추이
  - 공사유형별(토목/건축/전기 등) 현황 분석 자료

#### 생활안전지도 API 연계
- **위치기반 안전정보**
  - 건설현장 반경 500m 내 위험시설물(가스배관, 고압선) 위치
  - 인근 응급의료기관/소방서 접근 경로 정보
  - 공사장 주변 교통혼잡도 예측 데이터

#### 건설사업정보시스템 OpenAPI
- **설계도서 연계**
  - 건축물 평면도/단면도 등 디지털 설계파일(DWG, PDF)
  - 공사감리보고서 주요 항목(공정관리, 자재검수)
- **시설물 유지관리**
  - 준공 후 5년간 유지보수 이력(수리일자, 내용)
  - 설비별 점검주기(엘리베이터, 소방시설)

#### 지역별 건설현장 API
- **착공현장 실시간 현황**
  - 공사장별 진행률(착공~준공 단계)
  - 환경측정 데이터(소음/진동/먼지 수치)
  - 주변 교통통제 계획 정보

## 🛠 기술 스택

- **Frontend**: Next.js 15, React 18, TypeScript
- **UI**: Tailwind CSS, shadcn/ui
- **상태관리**: React Hooks
- **데이터**: 공공데이터포털 API, 로컬 스토리지
- **빌드도구**: Turbopack

## 🔑 API 설정

### 공공데이터포털 API 키 설정

실제 공공데이터를 조회하려면 API 키가 필요합니다:

1. [공공데이터포털](https://www.data.go.kr) 접속 및 회원가입
2. 다음 API 신청:
   - **건축물대장 표제부 조회 서비스**
   - **건축인허가 정보 서비스**
   - **KISCON 건설업체정보 서비스**
   - **생활안전지도 API**
   - **건설사업정보시스템 OpenAPI**

3. 프로젝트 루트에 `.env.local` 파일 생성:
```bash
NEXT_PUBLIC_DATA_API_KEY=여기에_발급받은_API_키_입력
```

4. 서버 재시작:
```bash
npm run dev
```

### API 연계 상태 확인

상세정보 팝업에서 각 API의 연계 상태를 실시간으로 확인할 수 있습니다:
- 🟢 **연결됨**: 실제 API 데이터
- 🟡 **시뮬레이션**: 테스트 데이터 (API 키 미설정 또는 연동 대기)
- 🔴 **실패**: API 호출 오류

## 🎯 사용법

### 1. 현장 검색
1. "전체 현장 조회" 버튼 클릭
2. 필터 조건 설정 (준공예정일, 상태)
3. 테이블 헤더 클릭으로 정렬

### 2. 타겟 현장 추가
- 검색 결과에서 "타겟 추가" 버튼 클릭
- 또는 수동 입력 탭에서 직접 등록

### 3. 상세 정보 조회
- "상세보기" 버튼 클릭
- 5개 탭으로 구분된 종합 정보 확인:
  - **KISCON**: 건설업체 정보 및 실적
  - **안전정보**: 위험시설물 및 응급시설
  - **건설정보**: 설계도서 및 감리보고서
  - **지역정보**: 공사진행률 및 환경측정
  - **통계**: 지역별 공사대장 통계

### 4. 데이터 다운로드
- CSV/Excel 형식으로 현장 목록 다운로드
- 필터링된 결과만 다운로드 가능
